# Territory Desk Redesign

## Goal

Refresh the La Ouirinance dashboard into a more distinctive, premium operational interface using the approved Coolors palette:

- `#4C5760` slate
- `#93A8AC` blue gray
- `#D7CEB2` parchment
- `#A59E8C` taupe
- `#66635B` warm gray

## Direction

Use a "Territory Desk" visual system: light mineral background, subtle map/grid texture, crisp command header, restrained cards, and strong information hierarchy for daily agency operations.

The interface should avoid generic SaaS dark gradients and oversized glass cards. It should feel like a premium dispatch table for door-to-door sales: readable, structured, and specific to sectors, vehicles, contracts, alerts, and rankings.

## Scope

- Update global theme variables and base surfaces.
- Restyle the app shell and navigation.
- Restyle shared UI primitives so all tabs inherit the new direction.
- Refine the dashboard view with better KPI hierarchy, section headers, and alert/readability treatment.
- Keep the existing data flow and behavior unchanged.

## Verification

- Run `npm run build`.
- Inspect the dashboard locally in desktop and mobile viewports.
- Confirm text remains legible and the layout does not overlap.
